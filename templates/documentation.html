Introduction

Resume Parser AI is a powerful tool that uses Natural Language Processing (NLP) and Machine Learning to analyze, categorize, and extract information from resumes. It can also match resumes against job descriptions to determine compatibility.

Key Features:
  • Extract detailed information from resumes (skills, experience, education, etc.)
  • Classify resumes into different categories
  • Match resumes with job descriptions and calculate compatibility scores
  • Simple REST API interface for easy integration

Installation and Setup

Prerequisites

Make sure you have Python 3.8+ installed on your system.

Step 1: Clone the Repository

git clone https://github.com/Soujash-123/ResumeParserAI.git
cd ResumeParserAI

Step 2: Install Dependencies

pip install -r requirements.txt
python -m spacy download en_core_web_sm
Note for Mac Users: If you encounter issues with PyMuPDF on Mac, you might need to install additional dependencies: 
brew install mupdf swig freetype

Step 3: Run the Application

python app.py

The server will start on http://localhost:5000 by default.

API Endpoints

POST /parse-resume

Extract detailed information from a resume PDF file.

Request

Content-Type: multipart/form-data

Parameter Type Description
file File The resume PDF file to parse

Response

{
  "contact_info": [...],
  "education": [...],
  "experience": [...],
  "skills": [...],
  "technologies": [...],
  "job_titles": [...],
  "companies": [...],
  "duration": [...]
}
cURL
Python
Node.js
curl -X POST \
  http://localhost:5000/parse-resume \
  -F "file=@/path/to/resume.pdf"
import requests

url = 'http://localhost:5000/parse-resume'
files = {'file': open('/path/to/resume.pdf', 'rb')}

response = requests.post(url, files=files)
data = response.json()
print(data)
const axios = require('axios');
const FormData = require('form-data');
const fs = require('fs');

async function parseResume() {
  const form = new FormData();
  form.append('file', fs.createReadStream('/path/to/resume.pdf'));
  
  try {
    const response = await axios.post('http://localhost:5000/parse-resume', form, {
      headers: form.getHeaders()
    });
    console.log(response.data);
  } catch (error) {
    console.error(error);
  }
}

POST /match-resume

Match a resume against a job description to calculate compatibility.

Request

Content-Type: multipart/form-data

Parameter Type Description
file File The resume PDF file to match
job_description Text The job description text to match against

Response

{
  "overall_match_score": 78.45,
  "text_similarity_score": 75.2,
  "skill_match_score": 85.7,
  "matching_skills": ["python", "machine learning", "sql"],
  "missing_skills": ["kubernetes", "aws"]
}
cURL
Python
Node.js
curl -X POST \
  http://localhost:5000/match-resume \
  -F "file=@/path/to/resume.pdf" \
  -F "job_description=We are looking for a Python developer with experience in machine learning and SQL..."
import requests

url = 'http://localhost:5000/match-resume'
files = {'file': open('/path/to/resume.pdf', 'rb')}
data = {
    'job_description': 'We are looking for a Python developer with experience in machine learning and SQL...'
}

response = requests.post(url, files=files, data=data)
result = response.json()
print(result)
const axios = require('axios');
const FormData = require('form-data');
const fs = require('fs');

async function matchResume() {
  const form = new FormData();
  form.append('file', fs.createReadStream('/path/to/resume.pdf'));
  form.append('job_description', 'We are looking for a Python developer with experience in machine learning and SQL...');
  
  try {
    const response = await axios.post('http://localhost:5000/match-resume', form, {
      headers: form.getHeaders()
    });
    console.log(response.data);
  } catch (error) {
    console.error(error);
  }
}

Example Workflow

A typical workflow for using the Resume Parser AI might look like this:

  1. Upload a resume to /parse-resume to extract detailed information
  2. Use /process-resume to classify the resume into a category
  3. Match the resume against a job description using /match-resume to determine compatibility
Tip: For batch processing of multiple resumes, you can create a simple script using the code examples provided above.

Troubleshooting

Common Issues

Issue Solution
API returns "No file provided" error Make sure you're sending the file with the key 'file' in your request
Error with PyMuPDF on Mac Install additional dependencies: brew install mupdf swig freetype
Spacy model not found Run python -m spacy download en_core_web_sm
Poor extraction results Make sure the PDF is properly formatted and text-based (not scanned)
